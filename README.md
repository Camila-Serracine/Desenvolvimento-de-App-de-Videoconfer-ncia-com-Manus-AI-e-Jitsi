# 🎮 Escape Game Digital

Aplicativo interativo de comunicação em tempo real com temática de escape room, desenvolvido com apoio do Manus AI e integração com o Jitsi.

---

## 📌 Sobre o Projeto

O Escape Game Digital é uma aplicação que combina videoconferência, chat e interação social, permitindo que amigos se conectem em salas privadas para se comunicar de forma dinâmica e imersiva.

O projeto tem como objetivo demonstrar, na prática, a integração de inteligência artificial com ferramentas modernas, acelerando o desenvolvimento de aplicações completas sem a necessidade de criar tudo do zero.

---

## 🚀 Funcionalidades

- 👤 Cadastro e login de usuários  
- 🏠 Criação e entrada em salas privadas  
- 💬 Chat em tempo real  
- 🎥 Videoconferência integrada  
- 👥 Visualização de participantes na sala  
- 🔊 Controle de câmera e microfone  
- 📱 Interface responsiva (mobile e desktop)  

---

## 🎨 Design e Experiência

- Tema inspirado em Escape Room  
- Interface em modo escuro (dark mode)  
- Paleta de cores com tons escuros e detalhes neon  
- Layout moderno, interativo e intuitivo  
- Elementos visuais com foco em imersão  

---

## 🧠 Tecnologias Utilizadas

- Manus AI  
- Jitsi (videoconferência em tempo real)  
- Integração de APIs  
- Estrutura preparada para expansão futura  

---

Este projeto foi desenvolvido com fins educacionais, com foco no aprendizado de:
